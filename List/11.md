## 문제

배열을 입력받아 output[i]가 자신을 제외한 나머지 모든 요소의 곱셉 결과가 되도록 출력하라.

 <table>
	<th>입력</th>
	<th>출력</th>
	<tr><!-- 첫번째 줄 시작 -->
	    <td>[1,2,3,4]</td>
	    <td>[24,12,8,6]</td>
	</tr><!-- 첫번째 줄 끝 -->
    </table>

## 코드

```python
from typing import List


class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        out = []
        p = 1

        for i in range(0, len(nums)):
            out.append(p)
            p = p * nums[i]

        p = 1

        for i in range(len(nums) - 1, -1, -1):
            out[i] = out[i] * p
            p = p * nums[i]

        return out
```
