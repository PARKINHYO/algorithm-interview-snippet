## 문제

정렬되어 있는 두 연결 리스트를 합쳐라.

 <table>
	<th>입력</th>
	<th>출력</th>
	<tr><!-- 두번째 줄 시작 -->
	    <td>1 → 2 → 4, 1 → 3 → 4</td>
	    <td>1 → 1 → 3 → 4 → 4</td>
	</tr><!-- 두번째 줄 끝 -->
    </table>

## 코드

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def mergeTwoLists(self, l1: ListNode, l2: ListNode) -> ListNode:
        if (not l1) or (l2 and l1.val > l2.val):
            l1, l2 = l2, l1
        if l1:
            l1.next = self.mergeTwoLists(l1.next, l2)
        return l1
```
